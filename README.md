# Jobs Helper (Boss & Liepin) - 求职助手

## 🚨 重要通告 (Important Notice)

**`Boss_helper.js` 已弃用 (DEPRECATED)**。

请切换使用 **`Universal_Job_Helper.js`**。该脚本现已全面支持 BOSS 直聘的所有功能，包括**图片简历自动投递**。

---

本项目是一个功能强大的油猴脚本（UserScript），旨在自动化“BOSS直聘”和“猎聘”两大招聘平台的求职流程，极大地提升求职效率。

## 核心功能

- **跨平台支持**: 一套代码同时支持 BOSS直聘 和 猎聘 网。
- **自动化流程**: 自动筛选职位、批量打招呼、自动投递简历。
- **支持图片简历自动投递 (BOSS直聘)**: 即使没有附件简历，也能自动发送图片简历。
- **智能筛选**: 支持通过关键词、薪资范围、公司规模等多种条件过滤不合适的职位。
- **可视化面板**: 提供一个悬浮的操作面板，可以实时监控运行状态、查看日志、动态调整配置。
- **健壮性设计**: 内置性能监控、错误处理和内存管理机制，确保长时间稳定运行。
