# 测试指南与报告

## 一、测试指南 (Testing Guide)

本项目包含两种测试方式：**自动化单元测试** 和 **手动集成测试**。

### 1. 自动化测试 (Jest)
用于验证核心工具函数和无副作用的业务逻辑。

- **运行所有测试**:
    ```bash
    npm test
    ```
- **运行监视模式 (开发时使用)**:
    ```bash
    npm run test:watch
    ```
- **生成覆盖率报告**:
    ```bash
    npm run test:coverage
    ```

### 2. 手动集成测试 (Manual Integration Testing)
由于油猴脚本高度依赖浏览器环境和目标网站的实时 DOM 结构，手动测试是必不可少的环节。

**测试步骤**:
1.  **准备环境**: 安装 Tampermonkey 插件，将最新脚本代码复制到新脚本中。
2.  **打开目标网站**: 访问 `zhipin.com` 或 `liepin.com` 的职位搜索页。
3.  **验证 UI**: 确认右下角/右上角出现了“招聘助手”悬浮面板。
4.  **功能验证**:
    - **筛选**: 在面板输入不常见的关键词（如"测试排除"），确认列表中的相关职位被跳过。
    - **启动**: 点击“开始运行”，观察日志是否滚动，页面是否自动翻页。
    - **停止**: 点击“停止”，确认脚本立即停止操作。
    - **配置**: 修改设置后刷新页面，确认设置值被保留。
